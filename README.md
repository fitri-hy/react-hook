# React Hook

This is Basic React Hook project  and React Routes using Tailwind CSS Framework.

## How to install
```sh
git clone https://github.com/fitri-hy/react-hook.git
```
```sh
npm install
```
```sh
npm start
```

## How to build
```sh
npm run build
```
